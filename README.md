# apartment1503-images

Public image host for the [Apartment 1503](https://instagram.com/apartment.1503)
Instagram project. Serves the Instagram-ready JPEGs (`0001.jpg` … `2994.jpg`,
chronological) over GitHub Pages so Instagram's Content Publishing API can fetch
each one by URL.

These are the prepped (resized/padded to Instagram spec) versions of images from
the OCTOBERS VERY OWN blog archive (2008–2021).
